Pelican, static site generator

Date Mon 16 May 2016 Tags Pelican / webdesign / static site generator / Python / Openlabs

Stasera grazie all'evento organizzato da Openlabs abbiamo presentato Pelican, un moderno static site generator scritto in Python.

Per lo sviluppo sul web utilizziamo una molteplicità di strumenti, scegliendo di volta in volta quello più adatto allo specifico compito. Negli ultimi tempi i generatori di siti statici, soprattutto per le considerazioni legate alla sicurezza, hanno guadagnato molto dello spazio prima riservato a CMS come Worpress, Joomla! o Drupal (che peraltro rimangono indispensabili in alcune specifiche circostanze). Nonostante l'aggettivo static i siti - grazie alle funzionalità introdotte da HTML5, CSS3 e Javascript - sono in realtà pienamente interattivi e rispondono alle interrogazioni dell'utente.

Questo stesso sito è realizzato in Pelican.
